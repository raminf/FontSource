FontSource — Privacy Policy
===========================

Last updated: April 2026

FontSource is a browser extension that detects fonts used on a webpage and shows
source and license hints. The extension does not run a FontSource account
system, analytics service, or cloud backend for storing your browsing activity.

1. What FontSource does NOT collect
-----------------------------------
FontSource does NOT collect or receive:

  • your name, email address, contacts, or precise location
  • your browsing history or a list of sites you visit
  • analytics, advertising identifiers, or crash reports sent to a
    FontSource-controlled server

2. What the extension accesses
------------------------------
When you use the extension on a page, FontSource may access:

  • the current webpage and its styles so it can identify fonts, font files,
    and related CSS information
  • public stylesheet URLs already referenced by that page, so the extension
    can fetch stylesheet text and parse @font-face rules when needed

These requests are made from your browser to the same third-party hosts the
page already uses. They are not sent to a separate server.

3. What is stored on your device
--------------------------------
FontSource stores only local extension settings in browser extension storage,
such as:

  • scan scope preference
  • preview and grouping preferences
  • your "find font" search URL template

This data stays in your browser storage. FontSource does not receive a copy of
it. If your browser syncs extension settings through your browser account, that
sync is controlled by your browser vendor, not by FontSource.

4. Network use
--------------
FontSource may make HTTPS requests from the extension background context to
fetch public stylesheet content referenced by the page you are scanning. This
is used only to improve font source detection. FontSource does not execute
remote JavaScript as extension code and does not upload scan results to a
FontSource service.

5. Changes
----------
If the extension's behavior changes, this file will be updated to match the
current code.
